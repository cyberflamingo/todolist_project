# TodoList

## DESCRIPTION:

A simple todolist project to learn how to package gem.

## FEATURES/PROBLEMS:

* Add todos!
* Remove todos!
* Add due date!
* Mark done/undone!

## DEVELOPERS:

After checking out the source, run:

```ruby
rbenv local 2.5.0
bundle install
bundle exec rake -T
```

This should get you started.
